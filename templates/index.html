Animal Picture API Documentation

Overview

Welcome to the Animal Picture API! This API allows you to retrieve and add pictures of animals such as dogs, cats, and capybaras. You can get a random picture of the specified animal or add new pictures to the database.

Endpoints

1. Get Random Picture

URL: /<animal>

Method: GET

Description: Retrieves a random picture of the specified animal.

Path Parameters:

  • animal (string): The type of animal. Must be one of dog, cat, or capybara.

Responses:

  • 200 OK: Returns a JSON object with the URL of the random picture.
  • 400 Bad Request: Returns an error if the animal type is invalid.
  • 404 Not Found: Returns an error if no pictures are available for the specified animal.
  • 500 Internal Server Error: Returns an error if there is a server-side issue.
curl -X GET "http://localhost:5000/dog"
{
    "url": "https://example.com/dog_picture.jpg"
}

2. Add Picture

URL: /add_picture

Method: POST

Description: Adds a new picture URL to the specified animal's database.

Request Body:

  • animal (string): The type of animal. Must be one of dog, cat, or capybara.
  • url (string): The URL of the picture to be added.

Responses:

  • 200 OK: Returns a success message if the picture is added successfully.
  • 400 Bad Request: Returns an error if the animal type is invalid or if the picture URL is missing.
  • 500 Internal Server Error: Returns an error if there is a server-side issue.
curl -X POST "http://localhost:5000/add_picture" -H "Content-Type: application/json" -d '{"animal": "dog", "url": "https://example.com/dog_picture.jpg"}'
{
    "status": "Thanks for that amazing picture! (More info at https://github.com/fatpotato317/API)"
}

Setup Instructions

Prerequisites

  • Python 3.6 or higher
  • Flask
  • Supabase
  • Decouple

Installation

1. Clone the repository:

git clone https://github.com/fatpotato317/API.git
cd API

2. Install the required dependencies:

pip install flask supabase-python python-decouple

3. Configure environment variables:

Create a .env file in the root directory and add the following lines:

URL=your_supabase_url
KEY=your_supabase_key

Running the API

To run the API, use the following command:

python app.py

Platform-Specific Instructions

Linux

# Install dependencies
sudo apt-get update
sudo apt-get install python3 python3-pip
pip3 install flask supabase-python python-decouple

# Run the API
python3 app.py

brew update
brew install python3

Windows

choco install python3
pip install flask supabase-python python-decouple

# Run the API
python app.py

Error Handling

The API returns appropriate error messages for invalid requests and server-side issues. Ensure to check the response codes and messages for debugging.
